# Análisis de Operadores ineficientes  - Call Me Maybe
Con el propósito de identificar a los operadores ineficientes del call center Call Me Maybe, se llevó a cabo el análisis de las métricas de cada operador que participó en esta evaluación. 

Para ello, identifiqué los promedios de tiempo de espera, llamadas perdidas, así como de ejecución de llamadas. También pude identificar patrones de comportamiento (días más activos, que tantas llamadas hacen entre operadores en comparación con las llamadas de clientes) de los operadores del call center. 

Con base en el análisis logré identificar a los operadores más ineficientes de cada una de las métricas establecidas. Asimismo, identifiqué que había complicaciones en la conformación de la base de datos, que jugaron un papel importante en el análisis. En ese sentido, considerando los resultados del análisis emití recomendaciones y líneas de acción para mejorar la eficiencia de los operadores, así como para que en futuros ejercicios se genere una base de datos más confiable y estable. 

Herramientas utilizadas: Python y Tableau
